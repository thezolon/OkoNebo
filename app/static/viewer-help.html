This page explains what you are seeing on the main dashboard and how to read it quickly.

Left Panel: Current Conditions

  • Location, current temperature, and condition icon reflect the latest active source.
  • Humidity, wind, pressure, visibility, dewpoint, UV, sunrise, and sunset update with each refresh cycle.
  • Source dots indicate feed health: green good, amber degraded, red failed, gray loading.
  • Active Alerts shows currently relevant weather alerts for configured monitored locations.

Center Panel: Radar + Hourly

  • Radar controls let you step, animate, pause, and jump to live frames.
  • Choose radar provider and optional overlay (precipitation, wind, temp, pressure, clouds).
  • 48-Hour Forecast can be viewed as a chart or table for quick trend reading.
  • Alert polygons can be toggled on/off for map clarity.

Right Panel: Planning + Diagnostics

  • 7-Day Forecast is pinned at the top for daily planning.
  • System Status shows sync timing, request counts, cache state, and upstream status.
  • Ops Timeline summarizes recent observations, alerts, and system events.
  • PWS Pair appears when configured and compares station conditions/trends.

Controls You Can Use

  • Units: switch between F/mph and C/km/h.
  • Full auto-refresh: keep data updating continuously.
  • Storm mode: higher refresh cadence during active weather windows.
  • Refresh Now: immediate pull of current feeds.

What To Do If Data Looks Stale

  • Check source dots and System Status to confirm upstream availability.
  • Use Refresh Now once to force a new fetch cycle.
  • If all providers are down, weather endpoints may return temporary 502 until fallback succeeds.
